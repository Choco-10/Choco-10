## 🤖 AI and ML Work

### 📄 DocuMind :- Document Intelligence System  
A system for asking questions over your documents.

Built using:
FastAPI, Celery, React (Vite), ChromaDB  

What it does:
- Upload and process documents
- Convert text into embeddings
- Retrieve relevant context using RAG
- Answer questions using LLMs
- Run heavy tasks asynchronously

It turns static documents into interactive knowledge sources.

---

### 📈 Stockify :- Stock Prediction System  
A stock analysis system with prediction and search capabilities.

Built using:
FastAPI, LSTM models, ONNX, yfinance, RapidFuzz  

What it does:
- Predicts next-day stock movement using ML models
- Provides fuzzy search over stocks and companies
- Serves trained models in ONNX format
- Includes a browser extension for quick access

An experiment in applying ML models inside real systems.
